# GW - تطبيق IPTV

تطبيق سطح مكتب لتشغيل القنوات والأفلام والمسلسلات عبر Xtream Codes API و M3U.

## المميزات

- 📡 تشغيل القنوات المباشرة (HLS / M3U8)
- 🎬 تصفح وتشغيل الأفلام
- 📺 تصفح المسلسلات بالمواسم والحلقات
- ⭐ نظام مفضلة منفصل لكل قسم
- 🔍 بحث وتصنيفات
- 🚀 تجاوز قيود CORS تلقائياً
- 🔒 دعم شهادات SSL الموقعة ذاتياً
- 🎨 واجهة عربية كاملة RTL

## متطلبات التطوير

- Node.js 18 أو أحدث: https://nodejs.org/
- npm (مرفق مع Node.js)

## التشغيل في وضع التطوير

```bash
# 1. ثبّت الاعتماديات
npm install

# 2. شغّل التطبيق
npm start
```

## بناء التطبيق للتثبيت

### للماك (macOS)

```bash
npm run build:mac
```

ينتج لك في مجلد `dist/`:
- `GW-1.0.0.dmg` - ملف التثبيت (سحب وإفلات)
- `GW-1.0.0-mac.zip` - نسخة مضغوطة

### للويندوز

```bash
npm run build:win
```

ينتج لك في مجلد `dist/`:
- `GW Setup 1.0.0.exe` - مثبّت كامل
- `GW 1.0.0.exe` - نسخة محمولة (بدون تثبيت)

### للنظامين معاً

```bash
npm run build:all
```

> **ملاحظة:** بناء نسخة الماك يحتاج جهاز ماك. بناء نسخة الويندوز يمكن من أي نظام.

## الأيقونات

ضع أيقونات التطبيق في مجلد `build/`:
- `icon.icns` للماك (1024x1024)
- `icon.ico` للويندوز (256x256)
- `icon.png` للينكس (512x512)

يمكنك توليدها من صورة PNG عبر:
- ماك: https://cloudconvert.com/png-to-icns
- ويندوز: https://cloudconvert.com/png-to-ico

## ملاحظات تقنية

- مبني على **Electron 28**
- يستخدم **hls.js** لتشغيل بث HLS
- `webSecurity: false` لتجاوز CORS من سيرفرات IPTV
- يقبل الشهادات الموقعة ذاتياً تلقائياً
